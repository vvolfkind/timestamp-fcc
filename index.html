My API Project #1: Timestamp microservice

How it works:
    1) You can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
    NOTE: you must follow one of these formats [Month Day, Year or Month Day Year] Year must be Text and Month must be an integer with two digits (e.g May is 05, December is 12)
    2) If it does, it returns both the Unix timestamp and the natural language form of that date.
    3) If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://timestamp-microservice-est.herokuapp.com/September%2025,%201912
https://timestamp-microservice-est.herokuapp.com/-1807228800

Example output:

{ "unix": -1807228800, "natural": "September 25, 1912" }
